# Twig API

## Parameters

### `name`

- Type: `string`

The filename of the icon, as found in the folder corresponding to the `@svg` alias.

### `ignore_missing`

- Type: `boolean`

Avoid empty `<span>` printing.

### `attr`

- Type: `array[]`

Additional attributes to add to the root element.
