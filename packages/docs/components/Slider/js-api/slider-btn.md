# SliderBtn

## Options

### `prev`

- Type: `boolean`
- Default: `false`

Goes to the previous slider item on click.

### `next`

- Type: `boolean`
- Default: `false`

Goes to the next slider item on click.

### `contain`

- Type: `boolean`
- Default: `false`

Disables the button when the slider is contained within its bounds.
