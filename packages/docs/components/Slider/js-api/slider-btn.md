# SliderBtn

## Options

### `prev`

- Type: `boolean`
- Default: `false`

Use this option to go to the previous slider item on click.

### `next`

- Type: `boolean`
- Default: `false`

Use this option to go to the previous slider item on click.

### `contain`

- Type: `boolean`
- Default: `false`

Use this option to disable the button when the slider is container within its bounds.
