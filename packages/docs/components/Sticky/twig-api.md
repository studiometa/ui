# Twig API

## Parameters

### `content`

- Type: `string`

Content for the sticky element.

### `attr`

- Type: `array`

Custom attributes for the root element.

### `inner_attr`

- Type: `array`

Custom attributes for the inner element.

## Blocks

### `content`

Sets the content of the sticky element. Defaults to the [`content` parameter](#content).
