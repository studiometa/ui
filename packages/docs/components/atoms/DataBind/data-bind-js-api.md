# DataBind JS API

The `DataBind` component can be used to keep a value in sync between multiple DOM elements.

## Options

### `prop`

- Type: `string`
- Default: `'textContent'`

### `name`

- Type: `string`
- Default: `''`

## Methods

### `set(value: string | boolean | string[])`

Set the value for the current instance.

### `get()`

Get the value for the current instance.
